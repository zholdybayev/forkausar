Тег form

Тег form это обязательный тег для передачи данных или написания какого-либо текста.

У тега form есть такие атрибуты как action и method.


Атрибут action указывает на назначение страницы где будут обработки всех данных которые были в теге form. Так как это другой язык программирования, мы пропускаем этот атрибут.


Атрибут method позволяет указать метод передачи данных из form. Есть значения как: get, post.


get - метод передачи данных по умолчанию, данные выписываются внутри URL адреса. Оно используется чаще чем post.


post - метод передачи данных, не выписывает данные внутри URL адреса.


Так как это связано с серверными языками, мы пропускаем эту часть, но в будущем если вы будете изучать серверные языки, вам это пригодится.


Если код не виден, вы можете скролить серый блок с кодом.


Например:
<form action="" method="get"></form>

Следующие теги являются дочерними элементами тега form.



Тег input

Тег input позволяет создать поле для ввода. Тег одинарный.

<form action="" method="get"> <input> </form>



У тега input есть не мало атрибутов. Рассмотрим все по очереди.

Атрибут maxlength указывает максимальное количество символов в поле.

<form action="" method="get"> <input maxlength="5"> </form>


Атрибут name указывает имя элемента input. Атрибут name используется для ссылки на элементы в JavaScript или для ссылки на данные формы после отправки формы. Следует прописывать этот атрибут для текстовых полей, таких как input или textarea. Про textarea мы еще поговорим.

<form action="" method="get"> <input name="input1"> </form>

Атрибут value позволяет сделать уже готовый текст в поле.

<form action="" method="get"> <input value="Hello, World!"> </form>


Атрибут placeholder позволяет сделать невидимый готовый текст в поле.

Например:
<form action="" method="get"> <input placeholder="Введите ваше имя"> </form>


Атрибут type указывает тип отображаемого элемента input. Есть множество значений:

<form action="" method="get"> <input type="text"> <!-- Текст --> <input type="password"> <!-- Пароль --> <input type="email"> <!-- Почта --> <input type="url"> <!-- Ссылка --> <input type="color"> <!-- Выбор цвета --> <input type="date"> <!-- Выбор даты --> <input type="time"> <!-- Время --> <input type="datetime"> <!-- Дата и время --> <input type="checkbox"> <!-- Галочки --> <input type="radio"> <!-- Галочка как кружочки --> <input type="file"> <!-- Загрузить файл --> <input type="button" value="Кнопка 1"> <input type="submit" value="Кнопка 2"> <input type="reset" value="Кнопка 3"> </form>

Очищение данных вводенных в полях

Я отключю свойство нажатия на эти три кнопки ибо сайт может багануться. Про тип submit мы поговорим на уроке JavaScript.


Если вы нажмете на галочку с типом radio, и потом попытаетесь нажать снова, то увидите что галочка не убралась. Такие проблемы случаются когда у нас два input поля с типами radio, и если мы нажмем на них двоих, то они оба будут нажаты.

<form action="" method="get"> <input type="radio"> <input type="radio"> </form>


Чтобы это исправить, надо обоим input полям с типом radio добавить атрибут name с одинаковыми значениями.

<form action="" method="get"> <input type="radio" name="name"> <input type="radio" name="name"> </form>



Также помимо атрибутов, есть еще и логические атрибуты. Это как бы атрибуты только без значений.

Логический атрибут hidden делает поле скрытым.

<form action="" method="get"> <input type="text" hidden> </form>


Здесь наверху на самом деле есть input поле. Просто оно скрыто засчет логического атрибута hidden.


Логический атрибут disabled делает поле неактивным и куда мы не можем ничего написать.

<form action="" method="get"> <input type="text" disabled> </form>


Логический атрибут readonly может только считывать символы в поле но мы не сможем вводить какой-либо текст. Но для этого, нам надо написать какой-то текст в атрибуте value.

<form action="" method="get"> <input type="text" value="Hello" readonly> </form>


Логический атрибут required, если в поле ничего не введено, данные не отправляются, то есть обязательное для заполнения. Для этого состояния нужна кнопка отправки данных.

<form action="" method="get"> <input type="text" required> </form>



Тег label

Тег label обычно прописывают вместе с тегом input. Тег label служит текстовой меткой для input. Пользователь благодаря ей может выбрать нужное текстовое поле кликом по тексту внутри тега label.

Структура выглядит так:

<form action="" method="get"> <label></label><input> </form>

Чтобы привязать текст в label в свое текстовое поле, внутри тега label надо прописать атрибут for. Значением же для этого атрибута послужит значение атрибута id в теге input.


Например:
<form action="" method="get"> <label for="someid">Нажми сюда</label><input id="someid"> </form>



Тег textarea

Тег textarea создает текстовую область как input только в него можно написать несколько строк текста. Следует тоже записывать в теге form.

<form action="" method="get"> <textarea></textarea> </form>


Если вам надо чтобы textarea находилась вне тега form, вы можете указать на форму, к которой относится текстовая область с помощью атрибута form. Значением же для этого атрибута послужит значение атрибута id в теге form.


Кроме этого, у него похожие атрибуты как у тега input такие как maxlength, name, placeholder но уже без value. Чтобы написать готовый текст в текстовом поле как это делает атрибут value, необходимо в самом теге написать этот текст.

<form action="" method="get"> <textarea>Hello, World!</textarea> </form>



Как вы наверно уже заметили, вы можете изменять ширину и высоту текстового поля. Чтобы это исправить надо задать свойство resize в языке CSS. Подробно мы пройдем всё это в языке CSS, но а пока можете просто посмотреть как это будет выглядить.


Чтобы сделать текстовое поле фиксированной по ширине и высоте, напишите в свойство resize значение none.

<form action="" method="get"> <textarea placeholder="Напишите что-либо"></textarea> <style> textarea { resize: none; } </style> </form>


Чтобы сделать текстовое поле фиксированной только по ширине, напишите в свойство resize значение vertical.

<form action="" method="get"> <textarea placeholder="Напишите что-либо"></textarea> <style> textarea { resize: vertical; } </style> </form>


Чтобы сделать текстовое поле фиксированной только по высоте, напишите в свойство resize значение horizontal.

<form action="" method="get"> <textarea placeholder="Напишите что-либо"></textarea> <style> textarea { resize: horizontal; } </style> </form>


После фиксированной текстовой области по ширине и высоте, вы можете добавить уникальные атрибуты как cols и rows.


cols - определяет ширину текстовой области. Ширина области рассчитывается в пикселях как ширина одного символа умноженная на число в значений.


rows - определяет высоту текстовой области. В качестве значения необходимо указать количество строк, которые будут видны без прокрутки текстовой области.


Например:
<form action="" method="get"> <textarea cols="20" rows="5" placeholder="Напишите что-либо"></textarea> <style> textarea { resize: none; } </style> </form>
