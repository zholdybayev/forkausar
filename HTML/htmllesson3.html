Тег table

Тег table позволяет создавать таблицы. У него есть объязательные теги tr и td. По желанию можно добавлять тег caption.

Тег tr - строки таблицы.

Тег td - колонны таблицы.

Тег caption - название таблицы.


Если код не виден, вы можете скролить серый блок с кодом.


Вот так выглядит сама структура таблицы:

<table> <caption></caption> <tr> <td>...</td> </tr> </table>

Давайте сделаем простенькую таблицу:

<table> <caption>Таблица</caption> <tr> <td>Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td>Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> <tr> </table>

Как мы видим, мы создали таблицу. Но согласитесь она какая-то странная. Чтобы исправить все недостатки, надо дописать атрибуты.


В тег table можно написать 4 атрибута: border, cellpadding, cellspacing, bgcolor.


Атрибут border задает количество границ таблицы. Внутри него надо писать числа.


Атрибут cellpadding задает отступы между границами таблицы и содержимым. Внутри него надо писать числа.


Атрибут cellspacing задает отступы между границами таблиц. Внутри него надо писать числа.


Атрибут bgcolor(в теге table) позволяет задать цвет всей таблице. Можно указывать название цветов таких как black, grey, yellow и т.д., или же указывать HEX цвета. Программисты почти везде чаще всего указывают HEX цвета. Пример HEX цветов: #000 - черный цвет, #ccc - серый, #fff - белый.


Например:
<table border="1" cellpadding="5" cellspacing="3" bgcolor="#dcdcdc"> <caption>Таблица</caption> <tr> <td>Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td>Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> </table>

Для тега tr тоже можно указывать атрибут bgcolor.

<table border="1" cellpadding="5" cellspacing="3" bgcolor="#dcdcdc"> <caption>Таблица</caption> <tr bgcolor="#fff"> <td>Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td>Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> </table>

В тег td можно написать 4 атрибута: colspan, rowspan, valign, bgcolor.


Атрибут colspan позволяет соединить горизонтальные ячейки. Внутри него надо писать число горизонтальных ячеек которые мы хотим соединить. Например давайте соединим ячейку с текстом "Заголовок" относительно двух нижних ячеек.

<table border="1" cellpadding="5" cellspacing="3" bgcolor="#dcdcdc"> <caption>Таблица</caption> <tr bgcolor="#fff"> <td colspan="2">Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td>Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> </table>

Атрибут rowspan позволяет соединить вертикальные ячейки. Внутри него надо писать число вертикальных ячеек которые мы хотим соединить. Например давайте Ячейку 2 соединим относительно боковых Ячеек 1 и 3:

<table border="1" cellpadding="5" cellspacing="3" bgcolor="#dcdcdc"> <caption>Таблица</caption> <tr bgcolor="#fff"> <td colspan="2">Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td rowspan="2">Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> </table>

Атрибут valign устанавливает вертикальное выравнивание содержимого ячейки. У него есть значения: middle, top, bottom, baseline.


middle - выравнивание содержимого по середине. Это значение по умолчанию.


top - выравнивание содержимого по верхнему краю строки.


bottom - выравнивание содержимого по нижнему краю строки.


baseline - выравнивание по базовой линии, при этом происходит привязка содержимого ячейки к одной линии.


Например:
<table border="1" cellpadding="5" cellspacing="3" bgcolor="#dcdcdc"> <caption>Таблица</caption> <tr bgcolor="#fff"> <td colspan="2">Заголовок</td> </tr> <tr> <td>Ячейка 1</td> <td rowspan="2" valign="baseline">Ячейка 2</td> </tr> <tr> <td>Ячейка 3</td> </tr> </table>
