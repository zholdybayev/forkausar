Тег button

Тег button создает кнопки. По своему виду он напоминает кнопку которую мы делали при помощи input. В отличии от того метода, здесь уже более расширенные возможности. Если при нажатии надо отправлять данные на сервер, то тег button надо помещать внутри тега form.


Если код не виден, вы можете скролить серый блок с кодом.


<button>Кнопка</button>



У него есть атрибуты type и form

У атрибута type есть значения button, submit, reset.


button - создает обычную кнопку.


submit - кнопка для отправки данных формы на сервер.


reset - кнопка для очистки введенных данных формы и возвращения значений в первоначальное состояние.



Атрибут form связывает между собой форму и кнопку если кнопка не находится в теге form. Значением же для этого атрибута послужит значение атрибута id в теге form.


Давайте сделаем маленький пример очистки введенных данных формы при клике:

<form action="" method="get"> <input type="text" placeholder="Ваше имя"> <input type="password" placeholder="Пароль"> <input type="email" placeholder="Email"> <button type="reset">Очистить данные</button> </form>



Тег audio

Тег audio создает аудио на странице. У него есть логический атрибут controls что позволяет управлять аудио. Про другие логические атрибуты вы можете поискать, т.к. это самый частоупотребляемый и основной логический атрибут этого тега. Данный тег используется редко, но все же о нем стоит знать.


У него есть обязательный тег source с атрибутами src и type.


src - адрес файла.


type - задает MIME-тип источника, а также аудио и видеокодек — так называется алгоритм хранения воспроизводимых данных. Атрибут неважен, можно не прописывать.

<audio controls> <source src="..." type=""> </audio>


Тег video

Тег video создает видео на странице. У него есть такой же логический атрибут controls что у тега audio. Данный тег употребляется чаще чем тег audio.


У него такой же обязательный тег source с атрибутами src и type. Точно такой же смысл как у тега audio.

<video controls> <source src="..." type=""> </video>
