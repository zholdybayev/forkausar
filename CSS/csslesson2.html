Свойство font-family

Свойство font-family позволяет изменить шрифт. Есть куча сайтов посвященные разным шрифтам, например шрифты Google - fonts.google.com. Значение inherit - наследует значение родителя. Свойство наследуется.

Шрифты можно подключать через тег link или через CSS правило @import. Мы будем делать при помощи @import. Выберите любой понравившийся шрифт, и скопируйте его import. CSS правило @import используется для импорта содержимого CSS файла в текущую таблицу стилей.

Затем вставьте этот import в самый верх CSS файла.


Чуть ниже вы можете обнаружить само свойство font-family с готовым значением.



Если код не виден, вы можете скролить серый блок с кодом.


Теперь можете просто вставлять его в нужные селекторы.

HTML код:
<body> <p>Я обычный шрифт</p> <p id="fonttext">У меня другой шрифт</p> </body>
CSS код:
#fonttext { font-family: 'Nunito Sans', san-serif; }

Свойство text-transform

Свойство text-transform позволяет изменить регистр символов. Есть значения как: lowercase, uppercase, capitalize, inherit. Свойство наследуется.


lowercase - каждый символ в нижнем регистре.


uppercase - каждый символ в верхнем регистре.


capitalize - каждый первый символ каждого слова в верхнем регистре.


inherit - наследует значение родителя.


Например:
HTML код:
<body> <p id="text-transform-1">Я в нижнем регистре</p> <p id="text-transform-2">Я в верхнем регистре</p> <p id="text-transform-3">Ну тут короче сами увидите</p> </body>
CSS код:
#text-transform-1 { text-transform: lowercase; } #text-transform-2 { text-transform: uppercase; } #text-transform-3 { text-transform: capitalize; }

Я в верхнем регистре

Свойство word-spacing

Свойство word-spacing позволяет изменить расстояние между словами. Значением же будут пиксели или отрицательные значения пикселей, значение inherit - наследует значение родителя. Свойство наследуется.

HTML код:
<body> <p id="word-spacing-1">Я короче длинный текст живищуй ради примера</p> <p id="word-spacing-2">У меня слова сближены к друг-другу</p> </body>
CSS код:
#word-spacing-1 { word-spacing: 30px; } #word-spacing-2 { word-spacing: -10px; }

Я короче длинный текст живищуй ради примера

У меня слова сближены к друг-другу



Свойство letter-spacing

Свойство letter-spacing позволяет изменить расстояние между символами. Значения же будут пиксели или отрицательные значения пикселей, значение inherit - наследует значение родителя. Свойство наследуется.

HTML код:
<body> <p id="texthello">Привет!</p> </body>
CSS код:
#texthello { letter-spacing: 10px; }

Привет!



Свойство line-height

Свойство line-height позволяет задать расстояния между строками. Значения же будут пиксели или отрицательные значения пикселей, значение inherit - наследует значение родителя. Свойство наследуется.

HTML код:
<body> <p>Привет</p> <p>Мир!</p> </body>
CSS код:
p { line-height: 30px; }

Привет

Мир!



Свойство text-decoration

Свойство text-decoration делает разное оформление текста такие как перечеркнутость, подчеркнутость и т.д. Так вот, значения: line-through, overline, underline, inherit. Элемент не наследуется.


line-through - перечеркнутый текст.


overline - линия проходит над текстом. На некоторых сайтах это значение не работает.


underline - подчеркнутый текст.


inherit - наследует значение родителя.


Например:
HTML код:
<body> <p>Я перечеркнут</p> </body>
CSS код:
p { text-decoration: line-through; }

Я перечеркнут
