Свойства width и height

Свойства width и height задают определенную ширину и высоту чему-либо. Значениями же будут единицы измерения такие как: px - пиксели, % - проценты относительно изначального размера и еще много разных единиц измерений. Свойства не наследуются.



Если код не виден, вы можете скролить серый блок с кодом.


Все блочные элементы по умолчанию имеют ширину 100% относительно родителя. Например давайте установим ширину 300 пикселей и высоту 100 пикселей для тега div(Добавлю границы блока чтобы вы могли увидеть его размеры):
HTML код:
<body> <div> <p>У меня ширина 300 пикселей, а высота 100 пикселей</p> </div> </body>
CSS код:
div { width: 300px; height: 100px; }

У меня ширина 300 пикселей, а высота 100 пикселей

!

Лайфхак

Если свойству line-height, которое мы уже прошли, написать высоту его родителя, то текст расположиться идеально по вертикали по центру.

Значение auto делает элемент на 100%, но само значение 100% не является значением auto. Разница между ними в том что значение 100% растягивает элемент по всему устройству даже если ширина устройства 99999 пикселей, а значение auto делает адаптивным 100% ширины элемента. Это значит что элемент будет растягиваться по текущей ширине, а не по основной как 99999 пикселей.


Когда ширина элемента задано как auto, ширина элемента ограничивающего содержимое блочного элемента, будет размер содержимого за вычетом ширины его частей как отступы(пройдем на четвертом уроке), толщина границы и т.д. А когда задано значение 100%, сам элемент будет занимать 100% ширины родителя, при этом отступы останутся.


С высотой height со значением auto все обстоит иначе. Высота элемента равна его содержимому, где значение по умолчанию auto.



Также существуют свойства max-width и max-height что позволяют задать максимальную ширину и высоту.

HTML код:
<body> <div> <p>У меня максимальная ширина 300 пикселей, а высота 100 пикселей</p> </div> </body>
CSS код:
div { max-width: 300px; max-height: 100px; }

У меня максимальная ширина 300 пикселей, а высота 100 пикселей

i

Примечание

В верхнем примере, когда у нас задан max-width вместе с max-height, max-width выполнится и задастся максимальная ширина, а поэтому высота подстроиться под контент автоматически, просто max-height будет работать "невидимо". Когда мы уже зададим определенную height высоту, контент уже не будет подстроен под автоматическую высоту, а задастся та высота которую мы указали в height. Значение height будет работать до тех пор пока не достигнет значения max-height. Значения выше уже не будут работать. Такой же принцип с width и max-width.

HTML код:
<body> <div> <p>У меня максимальная ширина все еще 300 пикселей, а высота 100 пикселей!</p> </div> </body>
CSS код:
div { max-width: 300px; width: 400px; /* Специально делаем значение выше максимального */ max-height: 100px; height: 200px; /* Специально делаем значение выше максимального */ }

У меня максимальная ширина все еще 300 пикселей, а высота 100 пикселей!



Точно по принципу max-width и max-height, существуют еще min-width и min-height. Они позволяют установить минимальные размеры ширины и высоты. Ниже уже быть не может.

HTML код:
<body> <div> <p>У меня минимальная ширина все еще 300 пикселей, а высота 100 пикселей!</p> </div> </body>
CSS код:
div { min-width: 300px; width: 200px; /* Специально делаем значение ниже минимального */ min-height: 100px; height: 50px; /* Специально делаем значение ниже минимального */ }

У меня минимальная ширина все еще 300 пикселей, а высота 100 пикселей!



Свойство background

Свойство background позволяет установить цвет или фоновое изображение на задний фон. Свойство background делится на несколько других свойств: 1) background-color, 2) background-image, 3) background-repeat, 4) background-size, 5) background-position, 6) background-attachment. Свойства не наследуются.


1) background-color

background-color позволяет установить цвет на фон. В этом же сайте цвет background-color для тега body является белый. Например давайте зададим синий цвет блоку:

HTML код:
<body> <div> <h2>Заголовок</h2> </div> </body>
CSS код:
div { height: 50px; background-color: aqua; text-align: center; }

Заголовок


2) background-image

background-image позволяет поставить фоновое изображение на фон. Структура свойства выглядит вот так:

... { background-image: url("URL или имя изображения"); }

После слова url в скобках надо писать либо URL изображения, либо путь к имени изображения.


Например давайте на задний фон блока поставим изображение природы(без указания ширины и высоты блока, изображение не появится):

HTML код:
<body> <div></div> </body>
CSS код:
div { background-image: url("../images/Nature.jpg"); width: 100%; height: 400px; }


Вот изображение появилось у нас на экране. Но согласитесь оно какое-то странное. Следующии свойства помогут исправить все недостатки.


3) background-repeat

background-repeat позволяет изменять повтор изображения. Есть значения как:


repeat-x - повторяется по горизонтали.


repeat-y - повторяется по вертикали.


repeat - повторяется в обоих значениях.


round - повторяется чтобы заполнить область фона не обрезаясь, если не получается, то изображение масштабируется.


no-repeat - размещается один раз и не повторяется.


Например:
HTML код:
<body> <div></div> </body>
CSS код:
div { background-image: url("../images/Nature.jpg"); background-repeat: no-repeat; width: 100%; height: 400px; }


4) background-size

background-size позволяет задать размер фонового изображения. Есть значения как:


...% ...% - в процентах. Первое значение ширина, второе высота.


...px ...px - в пикселях. Первое значение ширина, второе высота.


Если задано одно значение, то второе по умолчанию будет 50%.


cover - масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока.


contain - масштабирует изображение с сохранением пропорций таким образом, чтобы изображение целиком поместилось внутрь блока.


Например:
HTML код:
<body> <div></div> </body>
CSS код:
div { background-image: url("../images/Nature.jpg"); background-repeat: no-repeat; background-size: contain; width: 100%; height: 400px; }


5) background-position

background-position задает расположение фонового изображения. Есть значения как:


...% ...% - в процентах. Первое значение расстояние относительно левого края, второе значение относительно вверхнего края.


...px ...px - в пикселях. Первое значение расстояние относительно левого края, второе значение относительно вверхнего края.


left - расположен слева


right - расположен справа


top - расположен сверху.


bottom - расположен снизу.


center - по центру.


right top - два значения. Первое по горизонтали(left, center, right), второе по вертикали(top, center, bottom).


Если задано одно значение, то второе по умолчанию будет 50%.


Например чтобы это увидеть давайте уменьшим размер фонового изображения:
HTML код:
<body> <div></div> </body>
CSS код:
div { background-image: url("../images/Nature.jpg"); background-repeat: no-repeat; background-size: contain; background-position: center center; width: 100%; height: 400px; }


6) background-attachment

background-attachment задает свойство будет ли прокручиваться изображение вместе с содержимым или нет. Есть несколько значении:


fixed - фоновое изображение зафиксировано.


local - если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон выходящий за рамки элемента остаётся на месте.


scroll - позволяет перемещаться фону вместе с содержимым.



Все эти значения свойств можно писать в одном лишь свойстве background через пробел:

... { background: background-image background-repeat background-size background-position background-attachment; }
