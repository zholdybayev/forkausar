Свойство display

Свойство display позволяет изменить параметры элементов на блочный, не блочный и т.п. Свойство не наследуется. Значений очень много, но большинство из них не поддерживаются некоторыми браузерами, да и используются редко. Мы же разберем самые нужные и употребляемые. Значения:


block - делает элемент блочным. Например не блочный тег span будет вести себя как блочный, переноситься на новую строку.


inline - элемент отображается как не блочный. Блочные элементы переносятся на новую строку и их содержимое показывается на новой строке, но inline позволяет содержимым блочных элементов начинаться с того места где закончились предыдущие элементы.


inline-block - оно делает блочный элемент который обтекается с другими элементами страницы словно не блочный. Его внутренняя часть является подобию блочного, а сам элемент не блочный.


none - временно удаляет элемент из документа. Занимаемое им место не резервируется и веб-страница формируется так, словно элемента и не было. Можно снова сделать его видимым при определенных условиях добавлением стиля display с другим значением.


inherit - наследует значение родителя.


Если код не виден, вы можете скролить серый блок с кодом.


Например:
HTML код:
<body> <div> <p>Блок 1</p> </div> <div> <p>Блок 2</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; background-color: #60c7d9; display: inline-block; }

Блок 1

Блок 2

Лайфхак

Если блоку у которого есть содержимое, добавить свойство display со значением inline-block, то ширина и высота блока будет подстроено под его содержимое. Например:

HTML код:
<body> <div> <p>Ого, прикольно</p> </div> </body>
CSS код:
div { border: 1px solid black; display: inline-block; }

Ого, прикольно


Свойство position

Свойство position устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице. Свойство не наследуется. Есть значения как: absolute, relative, fixed, inherit.


absolute - позиционирует элемент относительно окна браузера в левом верхнем углу если у родителя нет свойства position. Если у родителя есть свойство position со значениями как absolute, relative или fixed, то элемент будет позиционироваться относительно верхнего левого края родителя.


relative - позиционировать объект относительно с того место где оно сейчас находится. Добавление свойств left, top, right и bottom изменяет позицию элемента и сдвигает его в ту или иную сторону от первоначального расположения.


fixed - похоже на значение absolute своим позиционированием относительно края окна браузера или родителя. Но отличие заключается в том, что элемент становится фиксированным при прокрутке страницы.


inherit - наследует значение родителя.


Также при надобности можно добавлять свойства самого position такие как top, left, right, bottom. Они указывают на направление сторон с которого будет позиционироваться элемент. Значения могут быть отрицательные и положительные.


Например:
HTML код:
<body> <div> <p>Привет, мир!</p> </div> </body>
CSS код:
div { border: 1px solid black; display: inline-block; padding: 10px; position: relative; left: 100px; }

Привет, мир!





Свойство z-index

Свойство z-index позволяет задать слой элемента. Если есть оси x и y которые являются соответственно шириной и высотой, то ось z, направленная в нашу сторону, является "уровнем высоты" элементов. То есть чем значение будет больше, тем элемент выше остальных. Значения же могут быть любые числа начиная с отрицательных и заканчивая положительными. Но мы рекомендуем писать с нуля, и так ведь чисел хватает по горло. Свойство не наследуется.


Это свойство работает исключительно если у элемента задано свойство position со значениями absolute, relative или fixed!



Например:
HTML код:
<body> <div id="blockunder">А я внизу остался</div> <div id="blockover">Я поверх него</div> </body>
CSS код:
#blockunder, #blockover { width: 100px; height: 100px; position: relative; display: inline-block; } #blockunder { background-color: silver; z-index: 1; } #blockover { background-color: blue; z-index: 2; left: -50px; }

Свойство float

Свойство float определяет по какой стороне будет выравниваться элемент, а остальные элементы будут обтекать его с других сторон. Не работает только на абсолютно позиционированных элементах. Свойство не наследуется. Есть значения как:


left - выравнивает элемент по левому краю, все остальные элементы обтекать его по правой стороне.


right - выравнивает элемент по правому краю, все остальные элементы обтекают его по левой стороне.


inherit - наследует значение родителя.


Например:
HTML код:
<body> <div id="block1"> <span>Я на правой стороне</span> </div> <div id="block2"> <span>А я обтекаю слева</span> </div> </body>
CSS код:
#block1, #block2 { display: inline-block; padding: 20px; } #block1 { background-color: aqua; float: right; } #block2 { background-color: silver; }

Я на правой стороне
А я обтекаю слева


Также по желанию, вместе со свойством float можно добавить еще одно свойство под названием clear.


Свойство clear

Свойство clear устанавливает с какой стороны запрещено обтекание другими элементами. Свойство не наследуется. Есть значения как: both, left, right.


both - отменяет обтекание элемента с левой и правой стороны. Следует писать это значение когда нужно снять обтекание элемента, но неизвестно с какой стороны.


left - отменяет обтекание с левой стороны. Все элементы которые были в левой стороне опустяться вниз и будут под текущим элементом.


right - отменяет обтекание с правой стороны.


Например:
HTML код:
<body> <div id="block1"> <span>Я на правой стороне</span> </div> <p>У меня отменено обтекание справа</p> </body>
CSS код:
#block1 { display: inline-block; padding: 20px; background-color: aqua; float: right; } p { clear: right; }

Я на правой стороне

У меня отменено обтекание справа



Свойство overflow

Свойство overflow управляет отображением элемента, если оно не помещается целиком. Есть значения:


visible - отображается весь элемент даже за пределами ширины и высоты. Это значение по умолчанию.


hidden - отображается только область внутри элемента, лишнии части элемента за пределами ширины и высоты будут скрыты.


scroll - добавляется полоса прокрутки.


auto - полосы прокрутки добавляются только при необходимости.


inherit - наследует значение родителя.


Например:
HTML код:
<body> <div id="blockparent"> <div id="blockchild"></div> </div> </body>
CSS код:
#blockparent { width: 250px; height: 250px; border: 2px solid grey; border-radius: 20px; overflow: hidden; } #blockchild { width: 100%; height: 100%; background-color: #242424; }



Свойство box-sizing

Свойство box-sizing применяется для алгоритма измерения ширины и высоты элемента.


Согласно спецификации CSS, ширины и высота блока состоит из размера самого содержимого(контента), внешний отступ(margin), внутрений отступ(padding), граница(border). Свойство box-sizing позволяет изменить этот алгоритм, задав размер не самого контента, а размер блока. Есть значения как:


content-box - стандарт CSS, width и height задает ширину и высоту контенту, значения внутренних отступов, отступов и границ не учитываются.


border-box - width и height включают в себя контент, значения внутренних отступов и границ, но не отступов(margin).


padding-box - width и height включают в себя контент, значения внутренних отступов, но не отступов(margin) и границ(border).


Например:
HTML код:
<body> <div id="block1">Контент 200 пикселей</div> <div id="block2">Контент, padding, border 200 пикселей</div> </body>
CSS код:
#block1 { width: 200px; height: 100px; margin: 0 20px; padding: 10px; border: 3px solid black; box-sizing: content-box; } #block2 { width: 200px; height: 100px; margin: 0 20px; padding: 10px; border: 3px solid black; box-sizing: border-box; }
