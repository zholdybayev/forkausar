Урок 1 CSS

Что такое CSS?


Введение в CSS

CSS — Cascading Style Sheets — каскадные таблицы стилей. То есть это язык, отвечающий за внешность HTML документа. Практический во всех сайтах используются в парах языки HTML и CSS.


Что же делает CSS? Всё просто, если HTML позволяет структурировать сам контент на странице, то CSS позволяет модифицировать его, сделав более красивым и привлекательным для пользователя.



Как же подключить CSS файл к HTML документу?

Вы уже знаете тег link из урока HTML. Так вот, файлы подключаются при помощи этого тега.


Если код не виден, вы можете скролить серый блок с кодом.


<link>

У этого тега есть обязательный атрибут href который задает местонахождения внешнего CSS-файла. Точно такой же принцип как у тега img. Мы указываем путь к CSS файлу.


Например давайте укажем путь к CSS файлу под названием style.css. Оно находится в той же папке что и HTML документ. Вы конечно же указывайте свой CSS файл.

<link href="style.css" rel="stylesheet" type="text/css">

Насчет этих других атрибутов, мы не будем зацикливать на них внимание. Просто знайте что они обязательны, можете копировать и вставлять в тег head.



Как же выглядит структура CSS кода?

Селектор { свойство: значение; }
Или
Селектор {свойство: значение;}

Именно вот так выглядит структура указания стилей и сама структура CSS кода. Мы рекомендуем писать первым вариантом. После значений обязательно надо ставить точку с запятой.


Но кроме того стили можно писать прямиком в HTML документе при помощи тега style.

<body> ... <style> ... </style> </body>

Но скажем честно, мы не рекомендуем писать таким образом. Ибо код станет еще длиннее, неудобнее и запутаннее. Лучше следует создать отдельный CSS файл куда мы будем уже записывать все стили. Но если вы хотите писать таким методом, то записывайте тег style в самом конце перед закрывающим тегом body. Это нужно для того чтобы браузер сперва прочитал все элементы HTML, а уже потом стили к ним.


Или можно писать атрибут style в самих тегах. Тогда стили будут приминены к этому тегу и не надо будет писать селектор.

<Тег style="свойство: значение;"></Тег>

Таким способом пишут только в тех случаях, если к элементу надо добавить 1-2 стиля. Но если в будущем в определенных условиях(например при адаптивности под мобильные устройства) вам надо будет изменить стили для элемента, то лучше не следует писать атрибут style в тегах.



Теперь давайте поговорим что такое селектор, свойство и значение.


Селектор - это часть CSS-правила, которая сообщает браузеру, к какому элементу (или элементам) веб-страницы будет применён стиль. Существует несколько основных видов селекторов к которым можно обращаться:


* - ко всем элементам.


Тег - ко всем таким тегам.


#id - при помощи идентификатора.


.class - при помощи классов.


Тег[атрибут="значение"] - селекторы на атрибут.


:Псевдокласс, ::Псевдоэлемент - псевдоэлементы.

i

Примечание

Почему перед названием id и class элементов ставится # и точка?

Потому что когда мы обращаемся к элементу по id как к селектору, перед самим названием id элемента ставится #.

#someid { свойство: значение; }

Точно такой же принцип с классом только перед названием класса ставится точка.

.someclass { свойство: значение; }

Про остальные виды селекторов мы узнаем по ходу урока.



Можно обращаться к нескольким селекторам через запятую.

Селектор, селектор, селектор { свойство: значение; }

Можно обращаться к дочерним элементам селекторов через пробел. Например давайте обратимся к тегу span в теге div:

div span { свойство: значение; }


CSS свойства - назначают любые визуальные стили, формируя таким образом уникальный внешний вид сайта. CSS свойств очень много. Некоторые свойства применяются только к определенным селекторам.



CSS значения - значения самих свойств CSS.

i

Примечание

Свойства для селекторов стоящих ниже других являются приоритетными. Например для два одиноковых селектора заданы свойства с одинаковыми значениями, и тому селектору который будет стоять ниже другого, выполнятся стили с его значениями.

Комментарий

Вы должны быть уже знакомы с комментариями из урока HTML. Он везде выполняют ту же роль. Комментарий в CSS пишут вот так:

/* Я комментарий */

Или в несколько строк:

/* Я длинный комментарий */

Свойства и значения текстов


И хочется сказать что некоторые свойства и их значения наследуются от родителя к дочерним элементам. Например если вы пропишите в тег body какие-то свойства, то это свойство распространится всем дочерним элементам тега body.



Свойство color

Свойство color задает цвет для текста. В значении можно использовать ключевые слова такие как red, blue, green и т.д., или же HEX коды цветов такие как #000 - черный, #fff - белый, inherit - наследует значение родителя. Свойство наследуется.


Например давайте обратимся к тегу span по его id и добавим свойство color:

HTML код:
<body> <span id="Hello">Привет!</span> </body>
CSS код:
#Hello { color: green; }

Привет!

Свойство text-align

Свойство text-align выравнивает текст по горизонтали относительно родителя. Есть значения как: left, right, center, justify. Свойство наследуется.


left - текст прижат слева. Это значение по умолчанию.


right - текст прижат справа.


center - текст по середине.


justify - текст прижат в обе стороны, слева и справа.


Например:
HTML код:
<body> <div> <span>Text center</span> </div> </body>
CSS код:
div { text-align: center; }

Text center


Свойство font-size

Cвойство font-size позволяет изменять размер шрифта. Есть несколько единиц измерения размера шрифта: px, %, em, rem. Если вы не указывали определенный изначальный размер шрифта, то в браузерах размер шрифта по умолчанию равен 14 пикселям. Значение inherit - наследует значение родителя. Свойство наследуется.


px - пиксели.

HTML код:
<body> <p>У меня размер шрифта 20 пикселей</p> </body>
CSS код:
p { font-size: 20px; }

У меня размер шрифта 20 пикселей



% - проценты относительно изначального размера шрифта.

HTML код:
<body> <p>У меня шрифт 150% относительно размера шрифта по умолчанию</p> </body>
CSS код:
p { font-size: 150%; }

У меня шрифт 150% относительно размера шрифта по умолчанию


Если изначальный размер шрифта будет больше или меньше, то указанный процент шрифта тоже будет больше или меньше относительно изначального шрифта.

HTML код:
<body> <div> Изначальный размер шрифта 120% <p>Размер шрифта 120% относительно 120%</p> </div> </body>
CSS код:
div { font-size: 120%; } div p { font-size: 120%; }

Изначальный размер шрифта 120%

Размер шрифта 120% относительно 120%



em - во сколько раз больше или меньше относительно изначального размера шрифта. Практически тоже самое что и проценты.


1em = 100%

HTML код:
<body> <div> Размер шрифта 16 пикселей <p>Я в полтора раза больше 16-ти пикселей</p> </div> </body>
CSS код:
div { font-size: 16px; } div p { font-size: 1.5em; }

Размер шрифта 16 пикселей

Я в полтора раза больше 16-ти пикселей

i

Примечание

В языках программирования, если мы хотим написать десятичную дробь, то вместо запятой пишется точка.


rem - относительно размера шрифта элемента html.

HTML код:
<body> <p>Половина размера шрифта от шрифта элемента html</p> </body>
CSS код:
html { font-size: 20px; } p { font-size: 0.5rem; }

Половина размера шрифта от шрифта элемента html



Свойство font-style

Свойство font-style позволяет изменить начертание шрифта на обычное, курсивное или наклонное. Значения: normal - обычное, italic - курсивное, oblique - наклонное, inherit - наследует значение родителя. Свойство наследуется.


Например:
HTML код:
<body> <p>Наклонный текст</p> </body>
CSS код:
p { font-style: oblique; }

Наклонный текст



Свойство font-weight

Свойство font-weight позволяет изменить жирность шрифта. Значением же надо указывать круглые трехзначные числа такие как 300, 500, 600 и т.п.(соответсвенно начинаются с тонких до жирных шрифтов), или же ключевые слова такие как lighter, normal, bold, bolder. Свойство наследуется.


Например:
HTML код:
<body> <p>Я жирнющий текст</p> </body>
CSS код:
p { font-weight: 900; }

Я жирнющий текст
