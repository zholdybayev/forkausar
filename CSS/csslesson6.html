Псевдоклассы


Псевдокласс - ключевое слово которое указывается после селектера через двоеточия ":". Псевдоклассы позволяют изменить стиль элемента при наведении, нажатии и т.д.


Если код не виден, вы можете скролить серый блок с кодом.


Селектор:псевдокласс { ... }

Псевдокласс :hover

Часто используемый псевдокласс который позволяет изменить стиль элемента при наведении.


Например:
HTML код:
<body> <div id="hoverblock"></div> </body>
CSS код:
#hoverblock { width: 200px; height: 200px; background-color: blue; } #hoverblock:hover { background-color: red; }



Псевдокласс :active

Псевдокласс active позволяет изменить стиль при нажатии.


Например:
HTML код:
<body> <div id="activeblock"></div> </body>
CSS код:
#activeblock { width: 200px; height: 200px; background-color: blue; } #activeblock:active { background-color: #313131; }



Псевдокласс :focus

Псевдокласс focus позволяет изменить стиль текстовым полям когда мы нажали на них и вводим что-то.


Например:
HTML код:
<body> <input type="text" placeholder="Введите что-то"> </body>
CSS код:
input[type="text"]:focus { border: 3px solid black; color: red; }



Псевдоклассы логических атрибутов

Вы наверно слышали в уроке HTML на теме текстовых полей про логические атрибуты такие как disabled, enabled, required, readonly, hidden. Так вот их можно писать как псевдоклассы.


Например:
HTML код:
<body> <input type="text" placeholder="Введите что-то" value="Привет!" disabled> </body>
CSS код:
input[type="text"]:disabled { background: black; color: white; }



Псевдокласс :empty

Псевдокласс empty позволяет изменить стиль элементам которые пустые.

HTML код:
<body> <div class="box">Я не пустой</div> <div class="box"></div> </body>
CSS код:
.box { width: 100px; height: 100px; margin-right: 50px; border: 2px solid black; } .box:empty { background: grey; }

Я не пустой


Псевдокласс :not

Псевдокласс not оно эквивалентно слову "кроме". То есть мы можем изменить стиль элементам кроме таких-то.


Например давайте в предыдущем примере мы изменим стиль НЕ пустым элементам:

HTML код:
<body> <div class="box">Я не пустой</div> <div class="box"></div> </body>
CSS код:
.box { width: 100px; height: 100px; margin-right: 50px; border: 2px solid black; } .box:not(:empty) { background: grey; }

Я не пустой


Псевдоклассы :first-of-type, :last-of-type

Эти псевдоклассы позволяют изменить стиль первому стоячему элементу или последнему.


Например:
HTML код:
<body> <div> <div class="divclass">Я первый</div> <div class="divclass">Я второй</div> <div class="divclass">Я третий</div> <div class="divclass">Я четвертый</div> </div> </body>
CSS код:
.divclass { float: left; border: 2px solid grey; } div .divclass:first-of-type { background-color: blue; }

Примечание

Псевдоклассы first-of-type и last-of-type работают когда мы указываем через родитель на дочерние теги как в верхнем примере. Потому что браузер не понимает сколько именно элементов и не видит их границы начало первого элемента и конца, а так как родитель ограничивает спектр этих элементов, он понимает где начало, первый элемент, и конец, последний элемент.

Псевдокласс :nth-child()

Псевдокласс :nth-child позволяет изменить стиль элементам на основе их нумерации. Есть несколько типов значений:


odd - все нечетные элементы.


even - все четные элементы.


число - определенное число соответствующее нумерации элемента.


3n-1 - 2, 5, 8 и т.д. элементы. n - натуральные числа.


Например:
HTML код:
<body> <div> <div class="divclass">Я первый</div> <div class="divclass">Я второй</div> <div class="divclass">Я третий</div> <div class="divclass">Я четвертый</div> </div> </body>
CSS код:
.divclass { float: left; border: 2px solid grey; } div .divclass:nth-child(3) { background-color: blue; }

Псевдоэлементы


Псевдоэлемент - ключевое слово которое указывается после селектора двумя двоеточиями "::". Оно позволяет изменить стиль к определенной части элемента.


Псевдоэлемент ::after

Псевдоэлемент after позволяет добавить невыделяемое содержимое после элемента. Для псевдоэлементов after и before, невыделяемое содержимое пишется в свойстве content внутри кавычек.

HTML код:
<body> <span>Стрелка вправо</span> </body>
CSS код:
span::after { content: "→"; }

Стрелка вправо

Псевдоэлемент ::before

Псевдоэлемент before позволяет добавить невыделяемое содержимое до элемента.

HTML код:
<body> <span>Внимание</span> </body>
CSS код:
span::before { content: "!"; padding: 1px 8px; background: #e8ee44; border-radius: 50%; margin-right: 10px; }

Внимание

Функция attr()

Функция attr() позволяет извлекать значения атрибутов элемента и применять его в стилях. Обычно оно используется для псевдоэлементов after и before и пишется в свойстве content только не в кавычках.


HTML код:
<body> <span howareyou="Как дела?">Привет!</span> </body>
CSS код:
span::after { content: attr(howareyou) ""; color: red; }

Привет!

Псевдоэлемент ::first-letter

Псевдоэлемент first-letter, по названию уже понятно, что оно позволяет изменить стиль первому символу в слове. Оно работает только на блочных элементах.

HTML код:
<body> <p>Привет!</p> </body>
CSS код:
p::first-letter { color: red; font-size: 24px; }

Привет!



Псевдоэлемент ::selection

Псевдоэлемент selection позволяет изменить стиль к части документа, который был выделен пользователем.

HTML код:
<body> <p>Выдели меня и произойдет нечто</p> <p>Выдели меня опять</p> </body>
CSS код:
::selection { /* Стиль для всех остальных элементов */ color: grey; background: #242424; } p:nth-child(2)::selection { color: white; background: grey; }

Выдели меня и произойдет нечто

Выдели меня опять
