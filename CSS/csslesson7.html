Flexbox | Флекс элементы


Что такое Flexbox?

Flexbox - это спецификация CSS в котором куча свойств позволяющие упростить некоторые задачи и проблемы при верстке сайта. Оно позволяет контролировать размер элементов, порядок и выравнивание элементов по вертикали и горизонтали, управлять свободным местом между элементами и т.п. Давайте рассмотрим его свойства.



display: flex;

Чтобы работать с Flexbox'ом, нужно сделать Flex-контейнер. Для этого нужно написать свойство display со значением flex. Такие элементы называются флексами. Оно выравнивает элементы по горизонтали:


Если код не виден, вы можете скролить серый блок с кодом.


HTML код:
<body> <div style="display: flex;"> <div class="divs"></div> <div class="divs"></div> <div class="divs"></div> </div> </body>
CSS код:
.divs { width: 33.333%; height: 150px; border: 1px solid grey; }



Свойство flex-direction

Свойство flex-direction задает выравнивание элементов по горизонтали и вертикали. Свойство не наследуется. Применяется к Flex-контейнеру. Есть значения как:


column - элементы распологаются вертикально и направленs сверху вниз.


column-reverse - элементы распологаются вертикально и напрпавленs снизу вверх.


row - элементы распологаются горизонтально и направлены слева направо. Значение по умолчанию.


row-reverse - элементы распологаются горизонтально и направлены справа налево.


Например:
HTML код:
<body> <div class="mainblock"> <div>Я первый блок</div> <div>Я второй блок</div> <div>Я третий блок</div> </div> </body>
CSS код:
.mainblock { display: flex; flex-direction: row-reverse; } .mainblock div { width: 33.333%; height: 150px; border: 1px solid grey; }

Свойство flex-basis

Свойство flex-basis определяет начальные размеры флекса. Похоже на свойство width. Лучше вместо него использовать flex-basis. Свойство не наслудется. Применяется к флекс элементам.

HTML код:
<body> <div class="mainblock"> <div>Я первый блок</div> <div>Я второй блок</div> <div>Я третий блок</div> </div> </body>
CSS код:
.mainblock { display: flex; flex-direction: row-reverse; } .mainblock div { flex-basis: 33.333%; height: 150px; border: 1px solid grey; }

Свойство flex

Свойство flex позволяет сделать флексы по ширине больше или меньше остальных относительно их Flex-контейнера. Значениями же принимаются натуральные числа. Чем больше число, тем флекс больше по ширине остальных. Если одному флексу задать свойство flex, то чтобы оно работало, надо остальным флексам задать это свойство. Свойство не наследуется. Применяется к флекс элементам.


Например:
HTML код:
<body> <div class="mainblock"> <div>Я первый блок</div> <div>Я второй блок</div> <div>Я третий блок</div> </div> </body>
CSS код:
.mainblock { display: flex; flex-direction: row-reverse; } .mainblock div { flex-basis: 33.333%; height: 150px; border: 1px solid grey; } .mainblock div:nth-child(1) { flex: 2; } .mainblock div:nth-child(2) { flex: 1; } .mainblock div:nth-child(3) { flex: 1; }

Свойство order

Свойство order позволяет изменить порядок флексов относительно их Flex-контейнера. Точно так же как свойство flex, значениями принимаются натуральные числа. Чем больше число, тем ближе оно расположено в конце. Если одному флексу задать свойство order, то чтобы оно работало, надо остальным флексам задать это свойство. Свойство не наследуется. Применяется к флекс элементам.


Например давайте чтобы это увидеть наглядно, уберем свойство flex-direction:
HTML код:
<body> <div class="mainblock"> <div>Я первый блок</div> <div>Я второй блок</div> <div>Я третий блок</div> </div> </body>
CSS код:
.mainblock { display: flex; } .mainblock div { flex-basis: 33.333%; height: 150px; border: 1px solid grey; } .mainblock div:nth-child(1) { flex: 2; order: 2; } .mainblock div:nth-child(2) { flex: 1; order: 3; } .mainblock div:nth-child(3) { flex: 1; order: 1; }

Свойство align-items

Свойство align-items позволяет сделать соответствующую высоту для блоков(убрать пустое пространство) и расположить их по вертикали в центр, вверх, вниз относительно боковых элементов. Свойство не наследуется. Применяется к Flex-контейнеру. Есть значения как:


flex-start - расположить флексы по вертикали в самый верх относительно самого большого элемента.


center - расположить флексы по вертикали в центр относительно всех элементов.


flex-end - расположить флексы по вертикали в самый низ относительно самого большого элемента.


stretch - значение по умолчанию.


Например:

HTML код:
<body> <div class="mainblock"> <div></div> <div></div> <div></div> </div> </body>
CSS код:
.mainblock { display: flex; align-items: center; } .mainblock div { width: 33.333%; border: 2px solid grey; } .mainblock div:nth-child(1) { height: 50px; } .mainblock div:nth-child(2) { height: 100px; } .mainblock div:nth-child(3) { height: 150px; }



Свойство justify-content

Свойство justify-content позволяет изменить положение флексов в ряду. Свойство не наследуется. Применяется к Flex-контейнеру. Есть значения как:


flex-start - флексы прижаты слева.


flex-end - флексы прижаты справа.


center - флексы по центру.


space-around - между флексов одинаковое расстояние.


space-between - флексы прижаты к левым и правым краям, между ними одинаковое расстояние.


Например:
HTML код:
<body> <div class="mainblock"> <div></div> <div></div> <div></div> </div> </body>
CSS код:
.mainblock { display: flex; align-items: center; justify-content: space-around } .mainblock div { width: 20%; border: 2px solid grey; } .mainblock div:nth-child(1) { height: 50px; } .mainblock div:nth-child(2) { height: 100px; } .mainblock div:nth-child(3) { height: 150px; }



Свойство flex-wrap

Свойство flex-wrap позволяет переместить флекс в нижний ряд при масштабировании. Свойство не наследуется. Применяется к Flex-контейнеру. Есть значения как:


nowrap - флексы выстраиваются в одну линию. Это значение по умолчанию.


wrap - флексы по правой стороне перемещаются в нижний ряд при масштабировании.


wrap-revese - флексы по левой стороне перемещаются в нижний ряд при масштабировании.
