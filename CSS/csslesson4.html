Стили для блоков


Свойство border

Свойство border позволяет задать границы блокам. Свойство не наследуется. Структура значения свойства border выглядит вот так:


Если код не виден, вы можете скролить серый блок с кодом.


... { border: ...px тип границы цвет; }

Первое значение - толщина границы, это свойство border-width.


Второе значение - тип границы, это свойство border-type. Типы границы:


Третье значение - цвет границы, это свойство border-color.


(Значение inherit - наследует значения родителя)


Но можно писать эти значения в перемешку.

Например:
HTML код:
<body> <div> <p>У меня тип границы solid!</p> </div> </body>
CSS код:
div { width: 200px; height: 100px; border: 3px solid green; }

У меня тип границы solid!



Чтобы добавить границу на одну из сторон, после свойства border надо добавить направление на любую из сторон через дефис: top - вверх, right - справа, bottom - снизу, left - слева:

HTML код:
<body> <div> <p>У меня граница только справа</p> </div> </body>
CSS код:
div { width: 200px; height: 100px; border-right: 3px solid green; }

У меня граница только справа



Чтобы убрать обводку с одной из сторон, после свойства border надо добавить направление на любую из сторон через дефис и добавить значение 0 или none. Направления сторон: top - вверх, right - справа, bottom - снизу, left - слева:

HTML код:
<body> <div> <p>У меня граница везде кроме справа</p> </div> </body>
CSS код:
div { width: 200px; height: 100px; border: 3px solid green; border-right: none; }

У меня граница везде кроме справа



Свойство margin

Свойство margin позволяет сделать внешний отступ элемента начиная от границы текущего элемента заканчивая внутренней границей его родителя. Довольно часто употребляемое свойство. Отступом является пространство от границы текущего элемента до внутренней границы его родительского элемента. Значения могут быть отрицательные и положительные. Если отступ отрицательный, элемент будет не отдаляться, а идти к тому месту с которого он относился. Свойство не наследуется. Значение inherit - наследует значение родителя.


Если мы пишем значения просто в свойство margin, то внешнии отступы будут в каждом направлении из сторон. Например:

HTML код:
<body> <div> <p>У меня отступ 50 пикселей слева, справа, сверху, снизу</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin: 50px; }

У меня отступ 50 пикселей слева, справа, сверху, снизу


Если мы укажем одну из направлении сторон через дефис, то внешний отступ будет задан в том направлении. Например давайте сделаем отступ сверху:

HTML код:
<body> <div> <p>У меня отступ 50 пикселей сверху</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin-top: 50px; }

У меня отступ 50 пикселей сверху



Можно задать два значения в свойстве margin. Первое значение будут отступы сверху и снизу, второе слева и справа.

... { margin: ...px ...px; }

Например:
HTML код:
<body> <div> <p>Короче вы поняли</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin: 20px 70px; }

Короче вы поняли



Можно задать 4 значения. По очереди устанавливает отступ для верхней, правой, нижней, левой стороны.


Например:
HTML код:
<body> <div> <p>Что написать? Щас сами все увидите</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin: 10px 0 30px 100px; }

Что написать? Щас сами все увидите



Можно выровнять элемент горизонтально по левому, правому, по центру при помощи значения auto. Если свойству margin-left задать значение auto, то элемент выравнается по правой стороне, так как auto занимает все доступное пространство.

HTML код:
<body> <div> <p>Эй, я здесь :)</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin-left: auto; }

Эй, я здесь :)


Точно так же если свойству margin-right задать значение auto, элемент выравняется по левой стороне:

HTML код:
<body> <div> <p>Теперь я тут :)</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin-right: auto; }

Теперь я тут :)


Чтобы выровнять элемент по центру, надо без направлении сторон написать просто margin с двумя значениями 0 и auto. Перве значение делает нулевые отступы сверху и снизу, а боковые стороны равны друг-другу:

HTML код:
<body> <div> <p>Я по середине :)</p> </div> </body>
CSS код:
div { width: 100px; height: 150px; border: 2px solid black; margin: 0 auto; }

Теперь я тут :)



Свойство padding

Свойство padding делает внутренние отступы между границами и содержимым элемента. Точно так же как margin, если мы пишем значения просто в свойство padding, то внутренние отступы будут в каждом направлении сторон. Свойство не наследуется. Значение inherit - наследует значение родителя.


Например:
HTML код:
<body> <div> <p>У меня внутренние отступы везде одинаковы</p> </div> </body>
CSS код:
div { border: 2px solid black; padding: 30px; }

У меня внутренние отступы везде одинаковы



Опять же как margin, если мы укажем одну из направлении сторон через дефис, то внутренний отступ будет задан в том направлении. Например давайте сделаем отступ слева:

HTML код:
<body> <div> <p>У меня отступ слева 50 пикселей</p> </div> </body>
CSS код:
div { border: 2px solid black; padding-left: 50px; }

У меня отступ слева 50 пикселей



Также можно писать сразу два значения. Первое будет отступ сверху и снизу, второе слева и справа:

... { margin: ...px ...px; }

Например:
HTML код:
<body> <div> <p>Короче щас сами всё увидите</p> </div> </body>
CSS код:
div { border: 2px solid black; padding: 20px 50px; }

Короче щас сами всё увидите

Можно задать 4 значения. По очереди устанавливает отступ для верхней, правой, нижней, левой стороны.


Например:
HTML код:
<body> <div> <p>У меня идеи закончились</p> </div> </body>
CSS код:
div { border: 2px solid black; padding: 10px 0 50px 50px; }

У меня идеи закончились



Свойство border-radius

Свойство border-radius позволяет установить радиус округление краев рамки или фона. Значения же могут быть пиксели, проценты и т.д. В случае с процентами, отсчет берется относительно ширины блока. Свойство не наследуется. Можно писать от одного до четырех значений.


1 значение - радиус указывается для всех четырех уголков.


2 значения - первое значение задает радиус верхнего левого и нижнего правого уголка, второе значение — верхнего правого и нижнего левого уголка.


3 значения - первое значение задает радиус для верхнего левого уголка, второе — одновременно для верхнего правого и нижнего левого, а третье для нижнего правого уголка.


4 значения - по очереди устанавливает радиус для верхнего левого, верхнего правого, нижнего правого и нижнего левого уголка.


Примеры:
HTML код:
<body> <div style="border-radius: 20px;" class="exampledivs"> <p>У меня одно значение в пикселях</p> </div> <div style="border-radius: 20% 30%;" class=exampledivs> <p>У меня два значения в процентах</p> </div> <div style="border-radius: 10px 30px 50px;" class="exampledivs"> <p>У меня 3 значения</p> </div> <div style="border-radius: 20% 0 20px 30px;" class=exampledivs> <p>У меня 4 значения</p> </div> </body>
CSS код:
.exampledivs { padding: 15px; border: 1px solid black; margin-bottom: 10px; }

У меня одно значение в пикселях

У меня два значения в процентах

У меня 3 значения

У меня 4 значения



Свойство opacity

Свойство opacity задает прозрачность какому либо объекту. Значения от нуля до единицы. Единица это первоначальная прозрачность объекта, ноль это значит что объект полностью прозрачен и невидим. Свойство не наследуется.


Например:
HTML код:
<body> <div> <p>У моего родителя на самом деле цвет черный</p> </div> </body>
CSS код:
div { height: 200px; background-color: black; text-align: center; opacity: 0.2; } div p { color: white; }

Мой родитель прозрачный, на самом деле он черный
