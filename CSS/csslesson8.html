Префиксы, тени, эффект перехода, состояния элемента


Что такое префиксы?

Префиксы - это непонятные слова которые стоят до свойств с двумя дефисами по сторонам. А если точнее, это небольшие дополнения для правил, которые помогают браузеру увидеть, что мы хотим использовать новое правило. Также префиксы необходимы для поддержки старых браузеров, в которых по умолчанию может не быть правила.


Если код не виден, вы можете скролить серый блок с кодом.


-webkit- -moz- -ms- -o-

Некоторые браузеры не понимают некоторых свойств, поэтому чтобы они поняли какое свойство и команду мы хотим задать, прописываются префиксы. Они означают, что свойство прописанное после префикса будет применяться исключительно в указанном браузере. Все остальные браузеры данное свойство будут игнорировать, т.к. для них данный префикс неизвестен.


Зачем и нужны префиксы и почему их пишут?

  1. Для включения в браузер свойств CSS, которые стандартом ещё полностью не реализованы и не утверждены.

  2. Для исправки проблем с кроссбраузерностью. Кроссбраузерность - это одинаковое отображение и работа сайта в различных браузерах. Если не учесть этот факт, то ваш сайт будет выглядить по разному на разных браузерах.

Префиксы браузеров

-webkit- : Chrome, Safari.


-moz- : Mozilla FireFox.


-ms- : Internet Explorer.


-o- : Opera.


Как определить, к какому свойству прописывать префикс?

На сайте Can I use вы можете прописывать свойства которые под сомнением поддержки браузерами, и оно покажет прямоугольники разных цветов которые говорят какой браузер поддерживает, к какому нужен префикс и у кого это свойство не реализовано.


Красный прямоугольник - браузер, в котором данное свойство не реализовано.


Зелёный прямоугольник с дефисом, расположенным в правом верхнем углу – браузер, в котором данное свойство используется через префикс.


Светло-зелёный прямоугольник – браузер, в котором данное свойство реализовано частично.


Зелёный прямоугольник – браузер, в котором данное свойство реализовано в соответствии со стандартом.


Как прописывать префиксы?

Давайте например возьмем свойство transition(пройдем чуть ниже). Для него нужны префиксы -webkit- и -o-. Префиксы прописываются следующим образом:

Селектор { -webkit-transition: ...; -o-transition: ...; transition: ...; }

Первые два свойства transition с префиксами работают в указанном браузере, а третье свойство transition без префиксов работает на всех браузерах где префиксы по умолчанию не требуются.



Свойство text-shadow

Свойство text-shadow позволяет сделать тень для текста. Прописывается 4 значения:

Селектор { text-shadow: 1) 2) 3) 4); }

1) Цвет тени.


2) Смещение по горизонтали. Положительное значение сдвигает вправо, отрицательное значение сдвигает влево.


3) Cмещение по вертикали. Положительное значение сдвигает вниз, отрицательное вверх.


4) Радиус размытия.


Например:
HTML код:
<body> <span>Привет мир!</span> </body>
CSS код:
span { color: #fff; text-shadow: black 0 3px 3px; }

Привет мир!


Свойство box-shadow

Свойство box-shadow позволяет сделать тень для блока. Свойство не наследуется. Есть 6 значений:


1) Ключевое слово inset. Оно делает тень внутри элемента. Можно прописывать по желанию.


2) Сдвиг тени по горизонтали. Положительное значение сдвигает вправо, отрицательное значение сдвигает влево.


3) Сдвиг тени по вертикали. Положительное значение сдвигает вниз, отрицательное вверх.


4) Радиус размытия тени.


5) Растяжение тени.


6) Цвет тени.


Например:
HTML код:
<body> <div> <span>Я тут короче текст видите</span> </div> </body>
CSS код:
div { width: 100px; height: 100px; padding: 10px 20px; text-align: center; border: 1px solid black; -webkit-box-shadow: 5px 5px 3px 0 grey; box-shadow: 5px 5px 3px 0 grey; }

Свойство transition

Свойство transition это эффект перехода между двумя состояниями элемента. Свойство не наследуется. Оно делится еще на несколько свойств:


1) transition-property

Свойство transition-property определяет список свойств(можно перечислять через запятые), участвующих в переходе. Значение по умолчанию: all(все свойства).


2) transition-duration

Свойство transition-duration задает длительность/время перехода. Значения: s - секунды, ms - милисекунды. 1s = 1000ms. Можно писать десятичные числа, например 0.5s без нуля: .5s.


3) transition-timing-function

Свойство transition-timing-function указывает как быстро по времени меняется указанное через transition-property значение свойства. Есть значения как:


ease - переход начинается медленно, разгоняется быстро и замедляется в конце. Или же можно писать функцию cubic-bezier(0.25,0.1,0.25,1). Это значение по умолчанию.


linear - переход происходит равномерно на протяжении всего времени, без колебаний в скорости. Его функция - cubic-bezier(0,0,1,1).


ease-in - переход начинается медленно, а затем плавно ускоряется в конце cubic-bezier. Его функция - (0.42,0,1,1).


ease-out - переход начинается быстро и плавно замедляется в конце. Функция - cubic-bezier(0,0,0.58,1).


ease-in-out - переход медленно начинается и медленно заканчивается. Функция - cubic-bezier(0.42,0,0.58,1).


4) transition-delay

Свойство transition-delay указывает задержку до эффекта перехода. Значения секунды - s, миллисекунды - ms. 1s = 1000ms. Значение по умолчанию - 0.



Можно всё это упростить и записать все эти значения свойств в одно свойство transition:

transition: 1) 2) 3) 4);


Например:
HTML код:
<body> <div> <span>Я тут короче текст видите</span> </div> </body>
CSS код:
div { width: 100px; height: 100px; padding: 10px 20px; text-align: center; border: 1px solid black; -webkit-box-shadow: 5px 5px 3px 0 grey; box-shadow: 5px 5px 3px 0 grey; position: relative; left: 0; -webkit-transition: 1s linear; -o-transition: 1s linear; transition: 1s linear; } div:hover { left: 100px; }

Примечание

Свойство transition пишется к основному первому состоянию элемента, в верхнем примере это был блок div. Также чтобы элемент перешел во второе состояние, первому основному состоянию надо задать первоначальные свойства, в верхнем примере это было left: 0;, второе состояние уже left: 100px;. Все значения transition работают и в обраную сторону, когда элемент дошел до второго своего состояния, и возвращается в свое изначальное состояние.

Свойство transform

Свойство transform позволяет трансформировать элемент, масштабировать, вращать, сдвигать, наклонать и комбинировать виды трансформаций. Свойство не наследуется. У этого свойства вместо значений функции. Есть функции как:


translateX(...) - сдвигает элемент относительно его обычного положения по оси X.


translateY(...) - сдвигает элемент относительно его обычного положения по оси Y.


translate(..., ...) - сдвигает элемент на новое место, перемещая относительно обычного положения вправо и вниз, используя координаты X и Y, не затрагивая при этом соседние элементы. Первое значение координата по оси X, второе по оси Y. Если нужно сдвинуть элемент влево или вверх, то нужно использовать отрицательные значения.


scaleX(...) - функция масштабирует элемент по ширине, делая его шире или уже. Если значение больше единицы, элемент становится шире, если значение находится между единицей и нулем, элемент становится уже. Отрицательные значения отображают элемент зеркально по горизонтали.


scaleY(...) - функция масштабирует элемент по высоте, делая его выше или ниже. Если значение больше единицы, элемент становится выше, если значение находится между единицей и нулем — ниже. Отрицательные значения отображают элемент зеркально по вертикали.


scale(..., ...) - масштабирует элементы, делая их больше или меньше. Значения от 0 до 1 уменьшают элемент. Первое значение масштабирует элемент по ширине, второе — по высоте. Отрицательные значения отображают элемент зеркально.


rotate(...deg) - поворачивает элементы на заданное количество градусов, отрицательные значения от -1deg до -360deg поворачивают элемент против часовой стрелки, положительные — по часовой стрелке. deg - degree - градус.


skewX(...deg) - деформирует стороны элемента относительно оси X.


skewY(...deg) - деформирует стороны элемента относительно оси Y.


skew(...deg, ...deg) - используется для деформирования(искажения) сторон элемента относительно координатных осей. Если указано одно значение, второе будет определено браузером автоматически.


Например:
HTML код:
<body> <div id="maindiv"> <div id="divtranslate"> <span>Я красиво двигаюсь</span> </div> <div id="divscale"> <span>Я становлюсь большим</span> </div> <div id="divrotate"> <span>Я кручусь</span> </div> <div id="divskew"> <span>Я параллелограмм</span> </div> </div> </body>
CSS код:
#maindiv { display: flex; justify-content: space-between; flex-wrap: wrap; } #maindiv div { height: 100px; padding: 10px 20px; text-align: center; border: 1px solid black; -webkit-transition: .5s; -o-transition: .5s; transition: .5s; } #divtranslate:hover { -webkit-transform: translate(10px, 10px); -ms-transform: translate(10px, 10px); transform: translate(10px, 10px); } #divscale:hover { -webkit-transform: scale(1.2); -ms-transform: scale(1.2); transform: scale(1.2); } #divrotate:hover { -webkit-transform: rotate(180deg); -ms-transform: rotate(180deg); transform: rotate(180deg); } #divskew:hover { -webkit-transform: skew(-20deg); -ms-transform: skew(-20deg); transform: skew(-20deg); }

Свойство transform-origin

Свойство transform-origin позволяет сместить центр трансформации, относительно которого происходит положения/размера/формы элемента. Обычно применяется для функции rotate(). Это свойство применяется со свойством и ключевыми кадрами animation и @keyframes(пройдем на следующем уроке). Писать два значения:


Первое значения - сместить центр трансформации по оси X. Значения могут быть left, center, right, единицы измерения.


Второе значения - сместить центр трансформации по оси Y. Значения могут быть top, center, bottom, единицы измерения.


Например:
HTML код:
<body> <div id="divrotate2"> <span>Я кручусь</span> </div> </body>
CSS код:
#divrotate2 { height: 100px; padding: 10px 20px; text-align: center; border: 1px solid black; transform-origin: right top; animation: rotatediv 5s linear infinite; } @keyframes rotatediv { 0% { -webkit-transform: rotate(0deg); -ms-transform: rotate(0deg); transform: rotate(0deg); } 100% { -webkit-transform: rotate(360deg); -ms-transform: rotate(360deg); transform: rotate(360deg); } }
