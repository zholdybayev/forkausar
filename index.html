О наших уроках

Эти уроки нацелены на изучение Frontend разработки. Существует два типа веб-разроботки, Frontend и Backend. Frontend это клиентская часть сайта, то есть то что пользователь видит на сайте, а Backend это серверная часть. Frontend состоит из языков HTML, CSS, JavaScript, а Backend из серверных языков таких как PHP, Python, Java и т.д. Ну что, начнем твой путь программиста?
