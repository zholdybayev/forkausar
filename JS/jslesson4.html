Всплывающие окна, Функции


Всплывающие окна это баннеры с уведомлениями, кнопками, полем для ввода. Рассмотрим всех по порядку.



Всплывающее окно alert

Всплывающее окно alert позволяет показать пользователю уведомление с текстом. Текст или другие выражения пишутся внутри скобок:


Если код не виден, вы можете скролить серый блок с кодом.


alert("Привет!");

Как обычно можно передавать значения переменных:

let word = 123; alert(word);

Всплывающее окно confirm

Всплывающее окно confirm позволяет показать уведомление с двумя кнопками "ОК" и "Отмена" или "Да", "Нет". На разных браузерах по разному, но суть прежняя.

let cont = "Продолжить?"; confirm(cont);

Всплывающее окно prompt

Это уже интересно. Всплывающее окно prompt позволяет пользователю вводить что-либо в текстовое поле. С этим можно уже делать реальные вещи по типу выводить то что написал пользователь в консоль, в окно и т.д. Вот:

prompt("Введите ваше имя");

Давайте узнаем имя пользователя. Сперва поместим prompt в переменную:

let name = prompt("Введите ваше имя"); alert("Привет, " + name + "!");

Не стоит просто так записывать alert, confirm, prompt без всяких вспомогательных объектов по типу переменных.


Как мы видим в верхнем примере, если ничего не вводить или нажать отмена, то происходит странные операции. Получаются пустые данные. Давайте мы это исправим при помощи проверок:

let name = prompt("Введите ваше имя"); if(name && name != null) { // Возвращает значение true(нажато "Да"), при этом поле не пустое alert("Привет, " + name + "!"); } else if(name === null) { // Возвращает значение false(нажато "Нет") alert("Вы нажали на Отмена"); } else { // Просто нажато "Да", при этом поле пустое alert("Вы не ввели имя"); }

Функции function

Функции это специальные участки кода, которые содержат в себе всякие команды. И при создании некоторых операции в JavaScript, мы можем ссылатся к этим функциями, и команды в этой функции будут выполнятся. Это удобно если например у нас имеется повторяющийся участок кода.


Чтобы создать функцию, используется ключевое слово function. Вот:

function имя функции(параметр) { ... }

Имя функции может быть любым но не начиная с цифр и не имея специальных символ. Любые придуманные названия в JavaScript, если в названии есть два слова, то начиная со второго слова, каждое последующее слово следует писать с большой буквы, например simpleFunction. Это этикет написания в JavaScript. Функция также принимает разные параметры. Название у них может быть любым. Параметры мы рассмотрим чуть позже.


Давайте создадим простую функцию:

function simple() { console.log("Я функция!"); }
Консоль

Почему консоль пустая? Дело в том что нам надо еще эту функцию вызвать. Вызывается она так:

function simple() { console.log("Я функция!"); } simple();
Консоль

Я функция!

Внутри параметров мы можем передавать разные значения и работать с ними:

function simple(element) { // 1) Есть параметр element console.log("Я " + element); // 3) Мы принимаем значение от параметра } simple("функция!"); // 2) Внутри параметра мы передаем значение+вызываем функцию
Консоль

Я функция!

Можно писать несколько параметров:

function simple(num, num2) { console.log(num + num2); } simple(22, 65);
Консоль

87

Нельзя создать функцию в другой функции. Но можно вызвать функцию в другой функции.



Локальный переменные - переменные записанные внутри функции. Она видна только в пределах функции, больше нигде она не видна + мы не можем обратиться к ней вне функции, если мы работаем внутри функции с переменной, то следует её писать в функции, то есть это будет локальная переменная.


Глобальные переменные - переменные записанные вне функции. К глобальной функции мы можем обращаться в функции и вне функции.


Разница let и var заключается в том, что переменная объявленная через var, видна везде в функции, в то время как переменная объявленная через let, видна только в рамках блока {...}, в котором объявлена.


Давайте создадим функцию которая будет подсчитывать суммы чисел в массиве:

function sour(arr) { let res = 0; for(let i = 0; i < arr.length; i++) res += arr[i]; // Берем элементы массива под индексом i и складывами все числа console.log(res); } let arr = [1, 2, 3, 4]; // Чтобы не путаться, название переменных можно записать как название параметров) sour(arr);
Консоль

10

Описание примера выше:

Мы передали параметр arr для функции, и чтобы не путаться с названиями переменных и параметров, мы создали переменную с названием параметра.


Внутри функции sour находится переменная res(result - результат), что будет показывать результат сложения.


Мы создали цикл for с длиной массива arr которого мы приняли от параметра arr.


Для переменной res мы плюсуем все элементы массива под каждым их индексом i.


Выводим переменную res с результатом в области функции. Результат доступен и виден только внутри функции.



Чтобы вернуть значение из функции, используется ключевое слово return. Давайте мы результат сложения, переменную res, вынесем за функцию при помощи ключевого слова return, и уже дальше поместим ее в другую переменную. Так мы сможем работать с ней вне функции:

function sour(arr) { let res = 0; for(let i = 0; i < arr.length; i++) res += arr[i]; return res; // Возвращаем значение из функции } let arr = [1, 2, 3, 4]; let abc = sour(arr); // Описание ниже

Описание: При вызове функции sour(arr) мы подсчитали сумму сложения чисел и поместили в эту переменную. Теперь sour(arr) как будто стал результатом res, и этот результат мы поместили внутрь переменной.


Теперь можно вывести эту переменную:

function sour(arr) { let res = 0; for(let i = 0; i < arr.length; i++) res += arr[i]; return res; } let arr = [1, 2, 3, 4]; let abc = sour(arr); console.log(abc);
Консоль

10
