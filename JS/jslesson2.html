Математические действия, Условные операторы


В JavaScript можно создать целый калькулятор. Но пока давайте рассмотрим базовые математические действия:


Сложение, вычитание, умножение, деление, остаток при делении

Сложение - +


Вычитание - -(минус)


Умножение - *


Деление - /


Остаток при делении - %


Можно записать в переменную все эти действия, например:


Если код не виден, вы можете скролить серый блок с кодом.


let number = 5 + 5; console.log(number);
Консоль

10

Можно записать числа в переменные, и уже потом делать математические операции:

let number1 = 63; let number2 = 7; console.log(number1 / number2);
Консоль

9

Если мы хотим изменить значение числа у переменной, можно прописать так:

let number1 = 22; number1 = number1 / 2; console.log(number1);
Консоль

11

Или можно поступить еще легче и просто добавить знак сложение, вычитание и т.д. перед знаком равно(=). Выглядит все следующим образом:

let number1 = 22; number1 /= 2; console.log(number1);
Консоль

11

Если мы хотим добавить или отнять единицу, то можно прописать два знака + или - после переменной. Вот так:

let number1 = 22; number1++; // Добавили единицу к 22 console.log(number1); let number2 = 43; number2--; // Отняли единицу от 43 console.log(number2);
Консоль

23

42

Давайте выведем математическую операцию в консоль с текстом:

let number1 = 45; let number2 = 23; console.log("Вычитание: " + number1 - number2);

Если мы попробуем ввести это и обновить страницу, вы увидите странное слово NaN. Оно значит, что число не определено. Потому что мы написали строку, потом добавили ему число, потом отняли число, и браузер не понимает что происходит в этом участке кода. Поэтому чтобы это исправить, нужно добавить скобки:

let number1 = 45; let number2 = 23; console.log("Вычитание: " + (number1 - number2));

То есть сперва выполнится то что в скобках, а уже потом то что снаружи.

Или же можно вместо скобок добавить запятую:

let number1 = 45; let number2 = 23; console.log("Вычитание:", number1 - number2);
Консоль

Вычитание: 22

Так наверно даже лучше, ибо число 22 не строка. Об этом прочитайте в примечании:

i

Примечание

Когда мы объединяем например строку и число, то число преобразуется в строку. Все строки объединяются с друг-другом.

Пример с объединением строк:

let numstr1 = "5"; let numstr2 = "5"; console.log(numstr1 + numstr2);
Консоль

55

Чтобы это исправить, надо добавить объект Number(). Оно превращает строку в число. Вот:

let numstr1 = Number("5"); let numstr2 = Number("5"); console.log(numstr1 + numstr2);
Консоль

10

Объект Math

Встроенный объект Math определяет свойства и методы для работы с математическими константами и функциями. С помощью него мы может высчитывать синусы, косинусы углов, находить значения числа пи, e и т.д.


Например давайте выведем число пи и e:
console.log("Число пи: " + Math.PI); console.log("Число e: " + Math.E);
Консоль

Число пи: 3.141592653589793

Число e: 2.718281828459045


Можно писать тригонометрию из 9-го класса такие как нахождение синус, косинус, тангенс, котангенс угла(в радианах) и т.д. Вот:

console.log("Синус 90 градусов: " + Math.sin(90) + " радиан");
Консоль

Синус 90 градусов: 0.8939966636005579 радиан

Также у него есть куча методов но мы рассмотрим только два, это методы max() и min().


Метод max() позволяет определить самое большое число из списка. Например:

console.log("Самое большое число: " + Math.max(6, 1, 9, 0, 30, 41, 12));
Консоль

Самое большое число: 41

А метод min() позволяет определить самое маленькое число из списка. Например:

console.log("Самое минимальное число: " + Math.min(6, 1, 9, 0, 30, 41, 12));
Консоль

Самое минимальное число: 0

Условные операторы


Операторы if, else

Условные операторы делают проверки разных условий и уже потом выполняют определенные действия. Есть операторы if, else if, else. Структура выглядит вот так:

if(Условие) { ... } else if(Дополнительное условие) { ... } else { // else это в крайнем случае ... }

Внутри круглых скобок if мы пишем условие, если оно не верное, его действие внутри фигурных скобок не выполнится и будет проверяться следующее условие в else if. Оператор else срабатывает в любом случае, если предыдущии операторы были неверны. Если бы предыдущие операторы были бы верными, то оператор else был бы просто проигнорирован. if и else if это соответственно проверки и дополнительные проверки, а else это не проверка а действие в крайнем случае. Оператор else не обязателен.


Операторы сравнения:

Равно - ==

Равно(с учетом тип данных) - ===

Больше - >

Меньше - <

Больше или равно - >=

Меньше или равно - <=

Не равно - !=


Давайте пропишем условия:

let num = 56; if(num != 10) { let num2 = "56 не равен 10" console.log(num2); } else if(num === 60) console.log(56 + " равен " + 60); else console.log("Ни одно условие не удовлетворено");
Консоль

56 больше чем 10

Если у нас одна строчка кода в условии, можно не прописывать фигурные скобки {}, а если несколько, то фигурные скобки обязательны.


Если мы хотим проверить переменную на правдивость(на значение true), то можно просто написать само название переменной в условие.

let bool = true; if(bool) { console.log("Это правда"); }
Консоль

Это правда

Если мы хотим проверить переменную на false, то можно перед названием переменной поставить восклицательный знак !.

переменной в условие.

let bool = false; if(!bool) { console.log("Это действительно ложь"); }
Консоль

Это действительно ложь

Можно проверить несколько условии словами И/ИЛИ.


И - &&


ИЛИ - ||


Например:
let num = 10; if(num != 10 || num >= 11) { // Если первое ИЛИ второе условие будет верным console.log("Не 10 и больше или равен 11"); } else if(num === 10 && num < 11) { // Если первое И второе условие будет верным console.log("Равно 10-ти и меньше 11-ти"); } else { console.log("Err"); }
Консоль

Равно 10-ти и меньше 11-ти

Оператор switch

Оператор switch проверяет равны ли значения случаям, а уже потом выполняет действия.


Сами случаи это слово case. Можно писать несколько case. Структура проверки значений выглядит вот так:

switch(Какую переменную будем проверять) { case проверка на значение: Действия; break; default: ...; // break }

Рядом с case мы пишем значение без скобок которое хотим проверить на равность. Слово break это закрывающий элемент который обязательно нужно писать в конце case.


default это аналогия с else. Если все предыдущие проверки не были верны, срабатывает действие в default. Для default слово break не обязательное, можно прописывать, можно нет.


Давайте напишем пример:
let str = "Hello"; switch(str) { case "World": console.log("Это слово World"); break; case 10: console.log("Это число 10"); break; case "Hello": console.log("Это слово Hello"); break; default: console.log("Err"); }
Консоль

Это слово Hello
