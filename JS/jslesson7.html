Скролл страницы, работа с классами


На прошлом уроке мы прошли глобальный объект window. Так вот для него можно весить обработчик события под названием scroll. Он позволяет делать операции при скроле страницы.


Чтобы узнать по оси Y, на какой высоте мы находимся, для window можно добавлять метод scrollY или же pageYOffset.


Например:

Если код не виден, вы можете скролить серый блок с кодом.


HTML код:
<body> <p id="posHeight"></p> </body>
JS код:
let posHeight = document.querySelector("#posHeight"); window.addEventListener("scroll", function() { let posY = window.scrollY; posHeight.innerHTML = posY; });



Работа с класами

Чаще всего добавление или удаление классов через JavaScript используется когда мы работаем со скролом страницы. Например на главной странице сайта ProgEasy, шапка сайта на высоте в 0 пикселях - прозрачная, и если уже больше нуля пикселей, то она белая. Это реализовано при помощи классов и CSS.


Для того чтобы работать с классами, используется метод classList. После мы пишем добавлять или удалять классы соответственно при помощи методов add() и remove().


Классы прописываются внутри скобок в кавычках.


Например:
HTML код:
<body> <div class="box"></div> </body>
CSS код:
.box { width: 100%; height: 150px; background-color: #3a64d8; } .box.active { /* Класс box с классом active */ background-color: #fff; border: 1px solid grey; }
JS код:
let box = document.querySelector(".box"); window.addEventListener("scroll", function() { let posY = window.scrollY; if(posY > 1560) // Не нужно писать ключевое слово px box.classList.add("active"); else if(posY <= 1560) box.classList.remove("active"); });


Проскрольте верх вниз чтобы увидеть результат блока сверху.


Про остальные виды скролов вы можете поискать в интернете, это был самый востребованный и часто употребляемый метод скрола.
