Вы наверно уже спросите, ну когда же мы будем взаимодействовать с HTML при помощи JavaScript? А этот урок настал. Мы разберем JavaScript атрибуты которые мы будем добавлять к HTML тегам и ключевое слово this.


Обработчики событии

Есть очень много атрибутов для HTML из JavaScript, но мы разберем те которые реально применяются на практике.


Атрибут onclick - при нажатии.


Атрибут onsubmit - при отправке формы.


Атрибут onmouseover - при наведении мышью.


Атрибут onmouseout - когда отвели мышку после наведения.


Атрибут onload - когда страница загрузилась(обычно добавляется к тегу body).


Атрибут onmousedown - когда нажали.

Атрибут onmouseup - когда отжали.


oninput - когда пользователь вводит что-то в input поле.

Примечание

При написании одной двойной кавычки внутри другой, оно будет написано неправильно и код не сработает. Для этого следует одну из кавычек писать одинарной.

Если при событии нужно выполнить несколько строчек кода, то лучше всего будет писать название функции в обработчиках событии, и уже потом совершать с ними операции.

HTML код:
<body> <button onclick="func()" class="button">Нажми на меня</button> <style> .button { padding: 5px; } </style> </body>
JS код:
function func() { let name = prompt("Введите ваше имя"); console.log(name); }

Ключевое слово this

Ключевое слово this в параметрах функции в обработчиках событии, говорит что мы как бы внутрь этой функции передаем объект(тег) с которым мы работаем. В JavaScript мы можем обращаться к параметру функции, и оно будет передаваться к текущему объекту(тегу).

HTML код:
<body> <Тег атрибут="название функции(this)"> </body>
JS код:
function название функции(параметр) { // Название параметра может быть любым ... }

Чтобы понять как работает this, давайте рассмотрим такой метод value. Он позволяет брать значения внутри объектов. Например давайте рассмотрим на примере input поля:

HTML код:
<body> <input oninput="inputPole(this)"> </body>
JS код:
function inputPole(event) { console.log(event.value); }

Консоль

Мы можем обращаться к атрибутам самого этого тега(не на примере input):

HTML код:
<body> <div id="box" onmouseover="boxMouseOver(this)"></div> <style> #box { width: 100px; height: 100px; background-color: #ae7db8; } </style> </body>
JS код:
function boxMouseOver(el) { console.log(el.id); }

Выбор объектов

Выбор объектов это наверно самая часто используемая вещь в JavaScript. Ведь при помощи этого мы можем выбирать объекты HTML и уже работать с ними.


Чтобы реализовать это, нам надо обратиться к такому глобальному объекту document. После мы указываем через метод, к какому объекту в HTML мы хотим обратиться, к объекту с тегом, к объекту с идентификатором, к объектам с классом.


getElementById() - выбор объекта по идентификатору.


getElementsByClassName() - выбор всех объектов по определенным классам. Не забывайте что в названии метода не element, а elementS.


getElementsByTagName() - выбор всех объектов по тегам. Тоже не element, а elementS.


Внутри скобок мы пишем кавычки и уже пишем значения:

HTML код:
<body> <div id="boxid"></div> <div class="boxclass"></div> <p>Я текст</p> </body>
JS код:
document.getElementById("boxid"); // Не нужно писать в кавычках символ # document.getElementsByClassName("boxclass"); // Не нужно писать в кавычках точку document.getElementsByTagName("p");

Будет лучше если мы запишем всё это в переменные:

let ident = document.getElementById("boxid"); let classes = document.getElementsByClassName("boxclass"); let tags = document.getElementsByTagName("p");


Или же можно выбирать объекты при помощи селекторов вот такими методами:


querySelector() - выбирает первый стоящий заданный объект.


querySelectorAll() - выбирает все объекты определенного типа.


Например давайте выбирем объект при помощи этих методов:

HTML код:
<body> <div id="boxident"></div> </body>
JS код:
let ident = document.querySelector("#boxident");

Как мы уже говорили, querySelector выбирает первый стоящий заданный объект. Например:

HTML код:
<body> <div class="boxclass"></div> <!-- querySelector выбрал первый стоящий объект --> <div class="boxclass"></div> <div class="boxclass"></div> </body>
JS код:
let classes = document.querySelector(".boxclass");

Давайте выберем все заданные классы при помощи querySelectorAll:

let classes = document.querySelectorAll(".boxclass");


После того как мы научились выбирать объекты на HTML странице, мы можем делать с ними всё что угодно. Сейчас мы рассмотрим что с ними можно делать:



Метод innerHTML

Метод innerHTML позволяет записать что-либо в объект(тег). Например:

HTML код:
<body> <p id="paragraph"></p> </body>
JS код:
let prgh = document.getElementById("paragraph"); prgh.innerHTML = "Привет!";

Привет!



Если мы хотим добавить метод с коротким кодом, не создавая переменные, то можно прямиком прописать вот так:

document.getElementById("paragraph").innerHTML = "Привет!";

Также внутри метода innerHTML можно писать HTML теги. Они также будут работать:

HTML код:
<body> <p id="paragraph"></p> </body>
JS код:
let prgh = document.querySelector("#paragraph"); prgh.innerHTML = "Я <strong>жирный</strong>";

Я жирный


Можно написать значения разных переменных, параметров, атрибутов при разных обработчиках событии. Тут уже играет ваша фантазия и надобность. Пример 1:

HTML код:
<body> <input onclick="onInputPole()"> <p id="paragraph"></p> </body>
JS код:
let prgh = document.querySelector("#paragraph"); function onInputPole() { prgh.innerHTML = "Введите что-то"; }


Пример 2:

HTML код:
<body> <input type="text" placeholder="Введите что-то" maxlength="20" id="inputtext"> <button onclick="btnClick()">Вывести</button> <p id="hereistext"></p> </body>
JS код:
let input = document.querySelector("#inputtext"); let p = document.querySelector("#hereistext"); function btnClick() { p.innerHTML = input.value; input.value = ""; }



Изменение стилей

При помощи JavaScript можно изменять стили к объектам в HTML. Для этого используется метод style. Этот метод позволяет изменить стили для объектов кроме текстов:

object.style.свойство = "значение";

Например:
HTML код:
<body> <div id="box"> <p>Привет я текст!</p> </div> </body>
JS код:
document.querySelector("#box").style.background = "#4a69f7";

Привет я текст!



Если мы пишем свойство у которого два слова в названии, то второе слово мы пишем с большой буквы:

document.querySelector("#box").style.backgroundColor = "#4a69f7"; // background-color --> backgroundColor


Метод cssText позволяет изменить стиль текстам:

object.style.cssText = "свойство: значение;";

Например:
HTML код:
<body> <p id="prghstyle">Привет я текст!</p> </body>
JS код:
document.querySelector("#prghstyle").style.cssText = "color: blue;";

Привет я текст!
