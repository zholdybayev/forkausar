Что такое JavaScript?


Введение в JavaScript

JavaScript - это динамический клиентский скриптинг. Если упростить то это язык который позволяет делать сложные вещи, функционалы страницы без перезагрузки страницы. Например при нажатии делалось что-то, при прокрутки страницы и т.д. На этом уроке мы не будем изучать профессиональный JavaScript, а базовый JavaScript. По своему уровню он запутаннее чем CSS, ибо тут есть странные моменты. При изучений JavaScript'а, му будем много работать с консолью, а позже разберемся как же работать с HTML документом через JavaScript.



Как и где начать писать JavaScript код?

JavaScript код можно прямиком писать в HTML документе в теге script. Вот:


Если код не виден, вы можете скролить серый блок с кодом.


<body> ... <script> ... </script> </body>

Или самым удобным методом это создать отдельный JavaScript файл(окончание должно быть .js) и подключить его в HTML документ. Подключается оно с помощью того же тега script при помощи атрибута src где мы указываем путь к JavaScript файлу. Вот:

<body> ... <script src="javascript.js"></script> </body>

Тег script с подключением файла через атрибут src следует писать в самом конце кода до закрывающего тега body.



В JavaScript нет конкретной структуры кода.



Комментарии

Комментарии в JavaScript бывают двух типов. Первый многострочный(хотя можно использовать как однострочный):

/* Я многострочный комментарий */ /* Я однострочный комментарий */

Второй однострочный:

// Я однострочный комментарий

Тег noscript

Этот тег нужен чтобы уведомлять пользователю если у него не включен JS:

<body> <noscript>Включите JavaScript!</noscript> </body>


Типы данных

В JavaScript существует несколько типов данных. Они деляется на примитивные и особенные типы данных.


Примитивные типы данных:


number - числа.

boolean - булевые выражения(true и false).

string - строки. Строки пишутся в одинарных или двойных кавычках. Разницы в них нету.

undefined - не определено.

null - пустота.


Особенные типы данных:

object - объект.



В примитивном типе данных boolean, кроме false есть еще и ложные значения такие как: "" - пустая строка, 0, NaN, null, undefined, []. Все они выдадут значение false.


О всех типах данных мы ознакомимся по ходу уроков.



Переменные

Переменные наверно самое главное в JavaScript. Оно используется почти всегда. Переменная это "ячейка" куда можно записать какую-либо информацию. Чтобы создать переменную, используются ключевые слова var, let или const. Структура СОЗДАНИЯ переменной выглядит вот так:


JS код:
var название переменной1; let название переменной2; const название переменной2;
i

Примечание

После заканчивания определенного участка кода в JavaScript нужно прописывать точку с запятой.

Ключевые слово var и let по сути одинаковы лишь с маленьким отличием(разберемся на уроке функции). Но var уже устаревшая форма создания переменной, так что следует использовать let


Ключевое слово const(константа) делает переменную где нельзя уже изменить информацию.



Чтобы записать информацию в переменную можно сделать так:

let hello; hello = "Привет";

Или же традиционным, удобным и быстрым форматом вот так:

let hello = "Привет";
i

Примечание

Названия переменных не должны повторятся.

Вывод информации

Теперь настало время поработать с консолью. Чтобы открыть консоль вы должны зайти сперва в код элемента страницы, затем в верхней части выбрать пункт "Console". Готово.


Есть несколько способов вывести информацию в консоль:

1) document.write(...);


2) console.log(...);


3) console.info(...);


4) console.error(...);


5) console.warn(...);


1) document - это глобальный объект всего который имеет доступ ко всему документу. Оно не выводит информацию в консоль, но выводит ее на странице. У него есть метод/функция write. Методы пишутся с точкой. Метод write может просто написать что-либо на странице. Но лучше не использовать его так как он не эффективен да и никто им не пользуется.


2) console.log - уже используют все для работы с консолью. Его используют для общего вывода журнальной информации. Именно его мы будем использовать в дальнейшем.


3) console.info - используют чтобы вывести необходимую информацию.


4) console.error - выводит сообщение о специальной ошибке.


5) console.warn - выводит предупреждающее сообщение. Это не ошибка, а просто предупреждение.


Чтобы увидеть результат в консоли, надо обновить страницу.


Давайте выведем что-то в консоль:

console.log("Привет, консоль!");
Консоль

Привет, консоль!

Консоль может сказать где и какая ошибка находится в коде. Она может показать в каком файле и на какой строчке кода ошибка. Также оно говорит какая именна ошибка произошла.


Можно выводить значения переменных:

let text = "Я текст"; console.log(text);
Консоль

Я текст

Чтобы изменить значение у переменной, можно прописать вот так:

let hello = "Привет"; hello = "Привет 2" console.log(hello);
Консоль

Привет 2

Можно изменить значение у переменной при помощи другой переменной:

let name = "Leha"; let name2 = "Alex"; name = name2; console.log(name);
Консоль

Alex

У директива const не получится уже изменить информацию. Она постоянна. Если мы попытаемся изменить значения у const, будет ошибка.

const hello = "Hello"; hello = "Bye"; // Будет ошибка

Чтобы узнать тип данных, используется оператор typeof. Например:

let num = 123; console.log(typeof num);
Консоль

number

Можно объединять строки, числа, переменные с помощью знака плюс +. Пример:

let name = "Имя"; let age = 18; console.log("Меня зовут " + name + " и мне " + age + " лет");
Консоль

Меня зовут Имя и мне 18 лет
